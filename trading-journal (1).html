📊 Google Sheets Szinkronizáció
Automatikus mentés Google Sheetre minden trade rögzítésekor. Ingyenes, csak Google fiók kell.
1 Apps Script létrehozása
Nyisd meg: script.google.com → Új projekt
Töröld ki az alapértelmezett kódot, és illeszd be az alábbi scriptet:
⚠️ A Sheet ID már be van égetve — nem kell módosítani!
// ═══════════════════════════════════════════ // TradeLog Pro — Bitcoin Google Sheets Sync // Sheet: https://docs.google.com/spreadsheets/d/17xbiFyCsM688fkwKLAC7cC1-TlB9is2FS_9hjvHHtJw // ═══════════════════════════════════════════ const SPREADSHEET_ID = '17xbiFyCsM688fkwKLAC7cC1-TlB9is2FS_9hjvHHtJw'; const SHEET_NAME = 'Bitcoin'; const HEADERS = [ 'ID', 'Dátum', 'Irány', 'Belépő ($)', 'Méret ($)', 'Tőkeáttét', 'Stop-Loss ($)', 'Take-Profit ($)', 'Fee ($)', 'Eredmény', 'P&L ($)', 'Indoklás', 'Tanulság', 'Rögzítve' ]; function doPost(e) { try { const payload = JSON.parse(e.postData.contents); const trades = payload.btc || []; const ss = SpreadsheetApp.openById(SPREADSHEET_ID); let sheet = ss.getSheetByName(SHEET_NAME); if (!sheet) { sheet = ss.insertSheet(SHEET_NAME); } // Ha üres a lap, fejléc hozzáadása if (sheet.getLastRow() === 0) { const hRow = sheet.getRange(1, 1, 1, HEADERS.length); hRow.setValues([HEADERS]); hRow.setFontWeight('bold'); hRow.setBackground('#1a2332'); hRow.setFontColor('#f7931a'); sheet.setFrozenRows(1); } // Meglévő ID-k beolvasása (ne duplikáljon) const lastRow = sheet.getLastRow(); const existingIds = lastRow > 1 ? sheet.getRange(2, 1, lastRow - 1, 1).getValues().flat().map(String) : []; // Csak új trade-eket fűz hozzá let added = 0; trades.forEach(t => { if (existingIds.includes(String(t.id))) return; const row = [ t.id || '', t.date || '', (t.dir || '').toUpperCase(), t.entry || '', t.size || '', (t.lev || '') + 'X', t.sl || '', t.tp || '', t.fee || '', t.resType === 'win' ? '✅ Nyerő' : t.resType === 'loss' ? '❌ Vesztes' : '⏳ Nyitott', t.pnl || '', t.reason || '', t.lesson || '', new Date().toLocaleString('hu-HU') ]; sheet.appendRow(row); // Zöld/piros sor szín eredmény szerint const newRow = sheet.getLastRow(); if (t.resType === 'win') { sheet.getRange(newRow, 1, 1, HEADERS.length) .setBackground('#0d2b1a'); } else if (t.resType === 'loss') { sheet.getRange(newRow, 1, 1, HEADERS.length) .setBackground('#2b0d12'); } added++; }); sheet.autoResizeColumns(1, HEADERS.length); return ContentService .createTextOutput(JSON.stringify({ status: 'ok', added: added, total: trades.length, synced: new Date().toISOString() })) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({ status: 'error', message: err.toString() })) .setMimeType(ContentService.MimeType.JSON); } } function doGet(e) { return ContentService .createTextOutput('TradeLog Pro API aktív ✓') .setMimeType(ContentService.MimeType.TEXT); } // Manuális futtatás teszteléshez — Script editorból indítható function testRun() { const ss = SpreadsheetApp.openById(SPREADSHEET_ID); const sheet = ss.getSheetByName(SHEET_NAME) || ss.insertSheet(SHEET_NAME); Logger.log('✅ Kapcsolat OK — Sheet: ' + sheet.getName()); }
2 Script kötése a Sheet-hez & Deploy
Az Apps Script editorban: Projekt neve → adj neki nevet (pl. "TradeLog")
Majd: Futtatás → testRun → Engedélyezd a hozzáférést (egyszeri Google login)
Ezután: Telepítés → Új deployment → Típus: Web app
Futtatás: Saját magam · Hozzáférés: Bárki
→ Deploy gomb → másold ki a Web app URL-t
